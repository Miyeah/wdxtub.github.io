看图说话

2015-07-07 | 分类于 Memory

这里有的，是一些图片，一些絮絮叨叨，以及很多回忆。

2015-06 在路上

在第一千零一次把头狠狠撞向青旅的天花板之后，不得已趴着开始一个一个字儿往手机里敲。一直不习惯触摸着打字，因为没有机械键盘琴键般敲击所带来的写作节奏，可又害怕下一秒就把所有的记忆抛诸脑后，于是我发现，浓度足够高的时候，本身就是一种节奏。

刚坐上离开上海的飞机就开始遗憾，可是正所谓所有的相见恨晚都是恰逢其时，有了遗憾，也就有了再去的理由。仔细想想，那些恨晚，究竟是在恨什么呢？也许命运无意间翻到了几年前的预约申请，于是有些敷衍地安排了一场相遇。小广场忽然听到的巨大烟花声响，好像放学的铃声，夜已深，有些时光终究留不住，有些瞬间永远忘不了。

一个满嘴跑火车的人，火车跑多了，自己有时都分不清真假。然而文字的真假，是容易分的。文字的力量，算是我对这个世界最后的信任。所以当顺着别人过去的脚步一路走来，感觉是奇妙的。就好像原本只有红色和蓝色的我，加入了一抹绿色，于是多了好多好多颜色。不比我的理性和坚硬，她感性与柔软的文字看起来很舒服，配上我从来没有接触过的音乐和异域风情，有种奇妙的穿越感，感觉用一种奇妙的方式参与了一小段过去，然而更希望有一大段未来。

长途旅行最苦恼的莫过于一开始就赌上了小小赌注却幸运地赢得了大奖，知道之后的旅途大概不会再有这样的好运。不知道自己属于乐观的悲观主义还是悲观的乐观主义，只是在心底会默默希望

如果在另一座城市能再有另一场邂逅，继续在不知名河边那些没说完的话，那该多好。

2015-06 复旦

分别的时候大家都意识到了下次再见不知道何年何月。认识醒醒好像是终于完成了一个少年最后的天真梦想，从此有了勇气独自上场去面对未知的一切。某些瞬间被无限拉长最终消失在所谓的回忆中，然而转化成另一种形式重新表现出来。

2015-06 无锡

旅行的惊喜在于，总会有意想不到变动，让你用一万种可能去和过去的自己告别，那些自己都不知道自己不知道的未知，和那些回过头来再看到的来时的路，都好像岔路口的路标那样渐渐从视线中消失，然而，有些东西留了下来。

2015-06 毕业

本科研究生六年过去，即将踏上新的旅程，感谢这片让我成长的土地，这里的人这里的事这里的美食这里的一切好的坏的，只有在即将离别的时候，才如此明显地迸发出来。

2015-03 苏州

道阻且长。有的时候，逃避反而是面对，因为已知道无处可逃。

2014-12 路

看着这张去亚丁路上的照片，仿佛回到了当时坐在马路上的瞬间。要很努力，才能看到更广阔的天地吧。

2014-10 二十年

一辈子有多少个二十年，一辈子又有多少段二十年的友情。我在，我们都在。

2014-07 珠海

还是想为梦想远行一次

2014-02 江边

蓝天白云快回来吧！连续好多天下雨都要发霉了！

想念和老爸一起骑车兜风的日子！

2014-02 发小兄弟

奈何难尽欢千日醉，此刻相对恨晚。 愿与你，尽一杯！ 聚与散，记心间！ 毋忘情义，长存浩气，日后再相知未晚。

和兄弟们在一起，仿佛又回到十多年前的教室操场小卖部。变的是容貌，不变的是情义。

2013-06 广州塔

骑车，恰逢恒大比赛，封路，故辗转江边，美景在前，唯有手机傍身，虽噪点多而依然照，聊以自慰。

2012-10 猴岛缆车

自猴岛坐缆车回来拍的，海南岛之旅最满意的一张风景照。

2012-12 夜景广州

飞机飞回广州的时候在市区绕了一下，得以拍下广州的夜景，从天上看小蛮腰还有珠江

2013-04 俯视平遥

五一前一个人去的平遥，街上稀稀拉拉的人群，人少才能优哉游哉地走走看看。拍照的地点是平遥最高的建筑。上去要五块钱，收门票的打野总是靠着墙和街坊吹水聊天。

2013-04 平遥小店

因为是下午四点到临汾的火车，所以一整天都消磨在驴友餐厅里了。店里几乎没有人，我就一边喝酒一边跟老板娘聊天。老板娘说醋和牛肉都是自家做的，怪不得这么好吃呢。
