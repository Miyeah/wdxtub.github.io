二十三岁的十八岁

2014-04-13 | 分类于 Memory

准备 GRE 考试的碎碎念

距离高考的那个夏天已经过去了四年又十个月，我却又怀念起了那段简单规律充实的日子。早上听着催命闹钟想在床上多赖一会儿，中午踩着铃声迎着老师严厉目光冲进课室，课上偷偷传纸条，晚自习肆意聊天打闹。三天，或者说两天半，当我真正要和这段日子说再见开始另一种生活的时候，起初兴奋，继而失落，最后习惯了新生活。

大学之后，学习就不再是往日的容颜，多少一夜速成少白头，韶华东流不复还。没想到反倒是读研的时候，又在托福和GRE身上找回了心底的十八岁。五年了，去过了大江南北，知了了人情冷暖；结识了不同的人，翻阅了各种的书；最后幸福地发现，那些年埋下的初心，真的还在。

于是我又过上了单调充实甚至有些乏味的日子，除了不得不开得会和不得不做的准备，要么背单词要么做阅读要么听听力要么看作文。多年前买了却没看的资料如今又派上了用场，一本一本搬，一页一页啃，终究是时间太少差距太大，这一个多月，我也算是尽力了吧。

当倒计时最终走到了零，考试结束的时候，我忽然意识到，其实生活从来不会给你时间和机会做好百分百的准备。无论成功或失败，没有准备好从来也绝对不应该成为为自己开脱的理由。放下可笑的所谓自尊，真正去接受失败。悲剧和喜剧从来都是不停转换的，如果从悲剧中吸取教训知耻后勇，那就是喜剧；但倘若沉溺于喜剧中不思进取，则难免悲剧。

都说不忘初心，但恰恰是这初心，最不被我们珍视。走得太快，错过太多风景，无尽的旅途，倘若一路如此，未免太无趣了。

广州的夏天就要来了，二十三岁的夏天，和十八岁的夏天，都是美好的夏天。
